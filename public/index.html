URL Shortner API Microservice

This is a URL Shortner microservice.

You can give it a URL and it will return you a shortened URL. If you hit the shortened URL it will take you to the original website See below for examples

  • https://little-url.herokuapp.com/new/https://www.google.com
  • https://little-url.herokuapp.com/new/http://foo.com:80

API will return you JSON data back and it will redirect you if you click the link below

  • https://little-url.herokuapp.com/2871
