URL Shortner API Microservice

This is a URL Shortner microservice.

you can pass it unix time or natural time and date

  • To give usnix time: http://url/1497225600000
  • To give natural time: http://url/June 12, 2078

API will return you JSON data back in following format:

  • {unixTime: 34323423232, naturalTime: 'February 02, 1971'}
